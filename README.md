# Beyond Presence Agent Project

This project integrates with the Beyond Presence API to create and interact with an AI agent.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Set up environment variables:
    - Create a `.env` file in the root directory.
    - Add your API key: `BEYOND_API_KEY=your_api_key_here`
    - (Optional) Set `API_BASE` if different from default.

## Usage

Start the server:
```bash
node server.js
```

List avatars:
```bash
node list_avatars.js
```
